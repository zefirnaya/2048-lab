# Игра 2048 - pink edition

Веб-версия классической игры 2048 с милой розовой тематикой.

## Особенности

### Основной функционал:
- Динамическая сетка 4x4
- Слияние плиток в 4 направлениях
- Система счета и таблица лидеров
- Сохранение состояния игры
- Отмена последнего хода
- Адаптивный дизайн для всех устройств
- Расслабляющая музыка и приятные звуковые эффекты

### Управление:
- **Десктоп**: стрелки клавиатуры (← ↑ → ↓)
- **Мобильные**: свайпы по экрану + виртуальные кнопки
